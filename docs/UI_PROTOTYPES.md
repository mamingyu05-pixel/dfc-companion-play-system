# DFC 关键页面原型

## Customer 首页

```text
[TopBar: DFC | 余额 | 登录状态]

[推荐陪玩横向区域]
  [推荐卡片] [推荐卡片]

[筛选栏]
  模式 / 段位 / 价格 / 在线 / 语音

[陪玩列表]
  [CompanionCard]
    头像 昵称 在线
    段位 / 擅长模式
    价格
    查看详情 / 下单

[MobileTabBar]
  首页 / 订单 / 充值 / 我的
```

## Customer 下单页

```text
[陪玩摘要]
[模式选择]
[时长选择]
[备注]

[价格明细]
  单价
  时长
  总价
  当前余额
  余额是否足够

[确认下单]
```

## Customer 充值页

```text
[余额卡片]

[充值步骤]
  1. 人工转账
  2. 上传截图
  3. 等待审核

[充值金额输入]
[上传截图]
[备注]
[提交充值申请]

[最近充值记录]
```

## Companion 工作台

```text
[TopBar: 在线状态切换]

[收益概览]
  可提现 / 待结算 / 今日订单

[当前订单]
  订单状态 / 模式 / 时长 / 操作

[待接订单]
  订单卡片列表

[MobileTabBar]
  工作台 / 订单 / 收益 / 我的
```

## Admin 数据看板

```text
[SideNav]
[TopBar]

[MetricCards]
  今日订单 / 待派单 / 待充值审核 / 待提现审核 / 投诉

[待处理队列]
  充值审核
  提现审核
  待派单
  投诉

[最近操作日志]
```

## Admin 派单页

```text
[待派单订单详情]
  客户 / 模式 / 时长 / 金额 / 备注

[陪玩筛选]
  在线 / 段位 / 模式 / 价格

[陪玩表格]
  昵称 / 状态 / 价格 / 当前订单数 / Discord绑定 / KOOK绑定 / 操作

[确认派单弹窗]
  订单
  陪玩
  通知平台：Discord + KOOK
  确认派单

[通知结果]
  Discord: SENT/FAILED
  KOOK: SENT/FAILED
```

## Admin 充值审核

```text
[筛选栏: 状态 / 时间 / 客户]

[充值申请表格]
  客户
  金额
  截图
  备注
  状态
  操作

[审核弹窗]
  截图预览
  金额确认
  通过 / 拒绝
  审核备注
```
