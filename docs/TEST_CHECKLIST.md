# 测试清单

## 认证与权限

- 客户只能进入 `/customer`。
- 陪玩只能进入 `/companion`。
- 管理员和超级管理员只能进入 `/admin`。
- 被封禁用户不能登录。
- JWT 过期后必须重新登录。

## 管理员

- 创建陪玩账号。
- 编辑陪玩资料。
- 重置密码。
- 封禁账号。
- 上架陪玩。
- 下架陪玩。
- 所有动作写入 `admin_logs`。

## 充值

- 客户提交充值申请。
- 管理员通过充值。
- 客户余额增加。
- 生成钱包流水。
- 管理员拒绝充值。
- 拒绝时余额不变。

## 订单

- 余额不足不能下单。
- 下单金额由后端计算。
- 客户可选择指定陪玩下单。
- 客户可选择平台代选下单。
- 平台代选订单必须由管理员人工派单。
- 陪玩列表展示语音偏好、语音风格和是否支持试音。
- 客户可申请试音，试音不改变订单为 `IN_PROGRESS`。
- 下单成功扣减客户余额。
- 派单成功后 Discord 通知。
- 陪玩接单成功。
- 重复点击接单只成功一次。
- 订单开始。
- 订单完成。
- 完成后平台抽成正确。
- 完成后陪玩收益正确。
- 每次状态变化写入 `order_status_logs`。

## 钱包

- 禁止负余额。
- 所有余额变化都有流水。
- 重复结算不会重复增加陪玩收益。
- 退款流程不会重复退款。

## 提现

- 陪玩可提交提现申请。
- 收益不足不能提现。
- 管理员审核通过。
- 人工打款确认完成。
- 提现完成生成流水。
- 拒绝提现释放冻结金额。

## Bot

- Discord 订单通知发送成功。
- KOOK 订单通知发送成功。
- Discord 充值通知发送成功。
- KOOK 充值通知发送成功。
- Discord 提现通知发送成功。
- KOOK 提现通知发送成功。
- Discord 投诉通知发送成功。
- KOOK 投诉通知发送成功。
- Discord 按钮接单回写 API。
- KOOK 按钮接单回写 API。
- Discord 和 KOOK 同时点击同一订单接单时，只能成功一次。
- 未绑定 Discord/KOOK 用户 ID 的陪玩不能接单。
- 错误的 `x-bot-token` 不能调用 Bot 内部接口。
- KOOK Webhook challenge 能正确返回 challenge。
- KOOK Webhook verify token 错误时拒绝请求。
- KOOK 语音房创建成功。
- KOOK 角色同步成功。
- Bot 异常时管理员频道收到提醒。
- Discord/KOOK 通知失败时写入 `bot_events.FAILED`，订单派单不回滚。

## 安全

- 密码加密存储。
- RBAC 生效。
- 管理员接口拒绝普通用户。
- 金额字段不能被前端覆盖。
- 并发接单安全。
- 并发结算安全。
- 异常回滚。
